Fingertracker 2016

A javascript library to capture fingertip locations using only the webcam in the browser (no OpenCV).

I worked to implement fingertip tracking in javascript for Ali Momeni's Dranimate project while working as a research assistant in ArtFab. Using background deletion and adaptive thresholding I was able to identify the contour of a hand, etc. I also wrote a color-tracking version in which users need to wear colored markers on their fingertips, which was more robust.
